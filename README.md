## 🌅 הוראות בוקר (30 שניות)

1. פתחו את הקישור למעלה ב-**Safari באייפד** (עובד בכל דפדפן)
2. **הגבירו את הווליום** 🔊 (ויש כפתור השתקה בפינה אם יונתן יתעורר לפני כולם 😉)
3. מומלץ: החזיקו את האייפד **לרוחב** (landscape)
4. הקישו על הכפתור הוורוד — והמוזיקה תתחיל
5. זהו! עונים על שאלה ← רואים את התשובה ← נהנים ממופע תמונות של דקה ← ממשיכים

**טיפים:**
- הקשה על המסך בזמן מופע תמונות מדלגת לשאלה הבאה
- בסוף כל 13 השאלות מחכה לכם הפתעה 🎬

## What's inside (for future Elik)

Self-contained static web app — no build, no frameworks. `index.html` +
vanilla JS/CSS. Hebrew RTL, optimized for iPad Safari.

- `js/data/questions.js` — the 13 questions (edit freely, redeploy = `git push`)
- `js/data/manifest.js` — generated photo manifest (per-question slideshows)
- `assets/photos/` — processed photos (resized, EXIF/GPS stripped)
- `assets/audio/` — music, transcoded to 112kbps
- Local dev: `python3 -m http.server 8137` and open http://localhost:8137
